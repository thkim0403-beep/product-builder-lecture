# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Project Outline

### Style and Design
*   Modern and clean design.
*   Responsive layout for mobile and web.
*   Visually appealing presentation of the generated numbers.

### Features
*   Generate 6 unique random numbers between 1 and 45.
*   Display the generated numbers to the user.
*   Button to trigger the number generation.

## Current Task

Create the initial version of the Lotto Number Generator application.

### Plan
1.  **`index.html`**:
    *   Set the title to "Lotto Number Generator".
